기본 속성 선택자

        - 요소에 작성된 속성을 이용해 선택하는 선택자

        - 속성, 속성 값이 일치하는 요소만을 선택

        [작성법]
        
            1) 같은 속성을 가지는 요소 선택
            [속성명] {css 코드;}

            2) 같은 속성, 속성 값을 가지는 요소 선택
            [속성명=속성값] {css 코드;}

            // 속성값은 '',""로 감싸져도 문제 없음

        바로 1 level 하위의 있는 요소를 선택하는 선택자
